    Blues 2.0

    Blues Front Framework v.2.0

    CSS and JS frameworks
    Purpose

    Blues built with pure css and javascript libraries from the scratch. It is cross browser and helps to build rich web interfaces for any websites.

    Requirements

    Pure CSS and Javascript written. No dependencies

    Size

    Blues lightweight

    • CSS (approx. 169* kb - not minimized)
    • JS (approx. 280* kb - not minimized)
    * - can change at any time.

    Repositories

    Feel free to use Blues

    • CDN on JSDeliver
    • Get it on Github
    • Teobyte repositories on Github
    Education

    Code of Blues framework written with the pure css and javascript, it is very simple and primitive, so you can easily use it for any educational reasons.
