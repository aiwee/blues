    Rating

    Star rating works along with bz-fonts only and icon named .bzicon-star-in-square

    And Blues.JS library rating bz-rating-2.0.0.js with a next code

    If you don't need to show rating value before the bars or stars, just remove parameter showvalue or set it to the false.

    By default rating works like a widget, to make it clickable add parameter rate: true.

    For a line rating add parameter type: 'line'. The line rating works with no bz-fonts linked to the page.
